Child Safety Standards Policy

Our application strictly prohibits any content related to child sexual abuse and exploitation (CSAE/CSAM).

We have a zero-tolerance policy for any user generated content that involves:
• Sexual content involving minors
• Exploitative behavior toward children
• Grooming, solicitation, or predatory behavior
• Illegal or harmful content involving minors

Reporting
Users can report content directly inside the application or via email.

Upon receiving a report:
• Content is reviewed immediately
• The account is suspended if violation is confirmed
• We cooperate with law enforcement when required
